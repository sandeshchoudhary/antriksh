# Antrisk
Open Library
 
### Dependencies
- Node: v8.12.0


### Setup Project 
```sh
$ git clone https://github.com/sandeshchoudhary/antriksh.git
$ cd antriksh
$ npm i
```

### Run production build
Runs application on PORT 3031 by default.
```sh
$ npm run build && npm run start
```

### Run in development mode
```sh
$ npm run dev
```

### Check errors
- Suite: Eslint
```sh
$ npm run lint
```

### Run test
- Test suite: Jest
```sh
$ npm run test
```
